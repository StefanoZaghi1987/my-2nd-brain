## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Create the initial `init_vault.py` file with the entry point, argument parsing, directory resolution logic, and color helper functions. This is the skeleton that all other subtasks extend.

Arg interface (matches init-vault.sh):
- No args → vault dir = `./second-brain-vault`
- Positional arg → explicit vault path
- `--here` → vault dir = cwd
- `--help` → print usage and exit

Directory resolution: `mkdir -p vault_dir`, then resolve to absolute path. `script_dir` = directory of `init_vault.py`.

Color helpers: `info()`, `ok()`, `skip()`, `warn()`, `err()` — use ANSI escape codes when `sys.stdout.isatty()` is True, plain text otherwise. Same visual style as init-vault.sh (info=blue, ok=green, skip=dim, warn=yellow, err=red).
<!-- SECTION:DESCRIPTION:END -->
